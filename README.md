# Dự Án PhotoFlyAI

Website giúp khách hàng tải ảnh chân dung lên và chuyển đổi thành ảnh thẻ AI 3x4 chuyên nghiệp chỉ trong vài bước đơn giản.

## 📦 Cài đặt dự án

*Clone dự án về

*Di chuyển vào thư mục dự án:

```bash
cd client
```

*Cài đặt các package cần thiết:

```bash
npm install
```

## 🚀 Chạy dự án

*Khởi động môi trường phát triển:

```bash
npm run dev
```
